# Changelog

## Version 1.0 Beta — Build 35

Initial public release.

### Features

- Clipboard speech playback
- Keyboard shortcut support
- Pause / Resume / Stop controls
- Voice audition and language selection
- Configurable default voice
- Automatic keyboard shortcut conflict handling
- Ctrl+Alt and Super+Alt fallback shortcut sets
- Safe uninstall support
- X11 clipboard support
- Wayland clipboard fallback support
- Install, reinstall, and uninstall support
- Desktop and menu launcher integration

### Tested On

- Linux Mint Cinnamon 22.2
- Linux Mint Cinnamon 22.3

### Notes

Linux Mint TTS originated from solving clipboard text-to-speech workflow issues while using Scrivener on Linux Mint.

The project evolved into a standalone clipboard-based TTS utility for Linux Mint Cinnamon.
